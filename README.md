# ServiceFlowEngine

**Professional field service management for small businesses.**

Built for cleaning companies, plumbers, landscapers, inspectors, and handyman services. Track jobs, manage teams, generate reports — all from your phone.

## Live Demo

- **Tailscale:** `http://100.103.198.30:9081`
- **Local:** `http://localhost:5080`

## Features

- Job estimation with time calculations
- Task tracking with timer system
- Photo documentation
- PDF report generation
- Performance analytics
- Email notifications (SendGrid)
- Mobile-optimized for field use
- White-label capable

## Tech Stack

- Flask + SQLAlchemy
- Bootstrap (mobile-first)
- Stripe subscriptions
- SendGrid email
- ReportLab PDF generation
- QR code integration

## Quick Start

```bash
pip install -r requirements.txt
python main.py
# Open http://localhost:5080
```

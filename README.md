## 🎯 What It Does

ServiceFlowEngine turns chaotic field service operations into a calm, profitable system:

- **📋 Job Estimation** — Time + material calculations with square footage support
- **⏱️ Task Tracking** — Built-in timer system. Know exactly where time goes.
- **📸 Photo Documentation** — Before/after photos attached to every job
- **📄 PDF Reports** — Professional reports generated instantly
- **📊 Performance Analytics** — See which jobs, customers, and services are most profitable
- **📧 Email Notifications** — Automated customer updates (SendGrid integration)
- **📱 Mobile-First** — Works perfectly on phones in the field
- **🏷️ White-Label Ready** — Your brand, your domain

## 🛠️ Tech Stack

- **Backend:** Flask + SQLAlchemy + SQLite/PostgreSQL
- **Frontend:** Bootstrap 5, mobile-first
- **Payments:** Stripe subscriptions
- **Email:** SendGrid
- **PDFs:** ReportLab
- **QR Codes:** python-qrcode

## ⚡ Quick Start

```bash
git clone https://github.com/ChaiLifeOTFT/ServiceFlowEngine.git
cd ServiceFlowEngine
pip install -r requirements.txt
python main.py
# Open http://localhost:5080
```
